# NERO Cover System

Use for report covers, PPT covers, section dividers, and short-video opening frames.

## Structure

- Top-left: source-aware eyebrow or category label.
- Main field: one precise title, not a slogan.
- Support field: date, period, scope, and evidence boundary.
- Visual field: coded chart, generated visual素材, or restrained texture.
- Footer: NERO Design Team, source boundary, and template status when relevant.

## Rules

- Use `color.brand.primary`, `color.brand.accent`, `color.surface.canvas`, and `color.text.primary`.
- Keep at least one clean overlay area for exact text.
- Do not let AI-generated visuals contain readable evidence text, figures, or chart labels.
- Avoid single-hue purple/blue gradients and decorative clutter.
- If the cover uses a generated visual, final text must be overlaid by Satori/Sharp, PptxGenJS, or Remotion.
